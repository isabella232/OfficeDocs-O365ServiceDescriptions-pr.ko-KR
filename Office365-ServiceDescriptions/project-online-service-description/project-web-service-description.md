# <a name="microsoft-project-for-the-web-service-description"></a>웹 서비스용 Microsoft Project 설명

Project 클라우드 기반 작업 및 프로젝트 관리를 위한 Microsoft의 최신 제품입니다. Project 웹용 응용 프로그램은 대부분의 요구와 역할을 충족할 수 있는 간단하고 강력한 작업 관리 기능을 제공합니다. Project 팀 구성원은 웹용 Project 사용하여 모든 크기의 작업을 계획하고 관리할 수 있습니다.

Project 대한 웹 사이트는 Microsoft Power Platform을 통해 구축됩니다. Power Platform은 PowerApps, Power Automate, Power BI 및 Microsoft Dataverse로 구성됩니다. Project 데이터 원본은 Microsoft Dataverse에 저장됩니다.

## <a name="available-plans"></a>사용 가능한 플랜

Project 웹용 웹 사이트는 Project 플랜 1, Project 플랜 3 및 Project 플랜 5. 웹 기능에 [Project 나열](/office365/servicedescriptions/project-online-service-description/project-online-service-description#project-for-the-web-features)된 기능은 웹 데이터 저장소에 대한 Project 데이터와만 상호 작용합니다. 사용자가 웹용 구독을 사용하도록 설정하는 Microsoft Project 자세한 내용은 프로젝트 관리 솔루션 및 비용 비교[를 참조하세요](https://www.microsoft.com/microsoft-365/project/compare-microsoft-project-management-software).

## <a name="learn-more"></a>자세한 정보

### <a name="microsoft-project-resources"></a>Microsoft Project 리소스

#### <a name="architecture"></a>아키텍처

[Project 아키텍처 다이어그램](/project-for-the-web/project-architecture-overview)은 웹용 Project Power Platform, Power BI 및 타사 서비스에서 작동하는 방법을 보여줍니다. 로드맵(포트폴리오 서비스)은 기본 환경에만 설치되는 반면 로드맵은 기본 앱이 지원되는 모든 환경의 Project 수 있습니다. 이 Project 솔루션은 기본, 프로덕션 및 샌드박스 Dataverse 환경에 설치할 수 있습니다.

#### <a name="project-roadmap-and-power-automate"></a>Project 로드맵 및 Power Automate

Project 로드맵을 사용하려면 Power Automate 구독의 일부로 프로비전되는 Project 필요합니다. Power Automate 함께 제공하는 Project 기능은 Flow Project 앱 섹션에 있는 Project 서비스 계획에 대한 Microsoft 365 관리 센터. 

#### <a name="project-for-the-web-and-microsoft-dataverse"></a>Project 및 Microsoft Dataverse에 대한 정보

Project 위해 Microsoft Dataverse를 사용하여 해당 데이터를 저장해야 합니다. Microsoft Dataverse 데이터베이스는 사용자 구독의 일부로 Project 프로비전됩니다. Microsoft Dataverse 기능에 대한 권한은 웹용 데이터를 지원하기 위해 데이터를 저장하고 Project 제한됩니다. Project 필요한 Microsoft Dataverse 기능은 Project 앱 섹션에 **Project 서비스 요금** 제로  Microsoft 365 관리 센터.

Project 5명 이상이 Project 라이선스가 있는 고객의 경우 웹용 Project 플랫폼 프로덕션 및 샌드박스 환경에 배포할 수 있습니다. [여기](/project-for-the-web/deploying-project)에서 자세히 알아보세요.

| 용량 포함/누적 | Project P1 | Project P3 | Project P5 |
|---------------------------|------------|------------|------------|
| Dataverse(이전의 일반 데이터 서비스) 데이터베이스: 포함/테넌트 | 3GB | 5GB | 5GB |
| Dataverse 데이터베이스: USL(사용자 구독 라이선스) | 50MB | 250MB | 250MB |

#### <a name="data-backup-and-retention"></a>데이터 백업 및 보존

Project 웹 서버의 데이터 백업 및 보존 정책은 Office 365. 자세한 내용은 [Data Retention, Deletion, and Destruction in Microsoft 365](/office365/Enterprise/office-365-data-retention-deletion-and-destruction-overview). 백업 및 복원을 관리하는 방법에 대한 자세한 내용은 [Back up and restore environments을 참조하십시오](/power-platform/admin/backup-restore-environments). 기본 환경은 삭제할 수 없습니다. 백업 및 복원을 지원하지 않습니다. 환경에 대한 자세한 내용은 [기본 환경을 참조하세요](/power-platform/admin/environments-overview#the-default-environment).
